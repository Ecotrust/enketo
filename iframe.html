Use a form inside an iframe

Request url by sending a POST request to:

http://enketo.formhub.org/launch/get_survey_url

with 2 variables:

server_url
form_id

Enketo will return a JSON response like this (+ some more properties): 

{
	success: true,
	iframe_url: "http://some.url/iframe",
	reason: null
}

The iframe_url will be the source for the iframe.
